## Usage
Download the "newvape.zip" folder, place it in your executors workspace, and unzip. Check the file for more instructions.

> [!NOTE]
> This **will not work** for low level executors like Xeno or Solara. Vape in general doesn't quite support these two either. [Get an executor](https://weao.xyz) with good sUNC and a good identity level (Synapse Z or Volt).
